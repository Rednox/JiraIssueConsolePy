# Tech Stack

Core Technologies:
- Python 3.9+
- pytest for testing
- ruff for linting
- mypy for type checking

Key Dependencies:
- httpx for async HTTP calls to Jira API
- pydantic for data modeling
- click for CLI interface
- pytest-asyncio for async test support
- pytest-watch for test watching

Development Tools:
- VS Code tasks for test/lint/type check
- pre-commit hooks
- Makefile for common tasks
