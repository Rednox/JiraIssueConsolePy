JiraIssueConsolePy is a Python port of the C# project `Jaegerfeld/JiraIssueConsole`. Its main purpose is to provide a command-line tool for:

1. Querying Jira issues using JQL filters
2. Calculating cycle times for issues 
3. Generating cycle time reports and analytics
4. Analyzing workflow transitions

The project is designed to help teams analyze their development workflows by providing insights into issue cycle times and flow metrics.
