# C# -> Python mapping (starter)

This file should be filled in during Phase 0 (Discovery). List each public C# class
and the Python module where it will be implemented.

Example:

- C#: Program.cs (Main)  -> `src/jira_issue_console/__main__.py` + `src/jira_issue_console/cli.py`
- C#: JiraClient.cs       -> `src/jira_issue_console/jira_client.py`
- C#: IssueTransformer.cs -> `src/jira_issue_console/core/issues.py`
